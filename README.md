### Compile sass

### Prerequisites

```bash
# Install (global/alias) sass
gem install sass
# Install node-sass from package.json
npm install
```

```bash
# Compile with
npm run build-sass
# Or, watch sass and compile
npm run watch-sass
```

### Preview:

```bash
python -m SimpleHTTPServer 8080 index.html        
```
